📊 Employee Attrition Analysis Using Excel

## 📖 Project Overview
This project focuses on analyzing employee attrition data to understand the key reasons behind employee turnover. The analysis is performed using Microsoft Excel by cleaning the data, creating meaningful metrics, and building an interactive dashboard.

The main goal of this project is to identify patterns in employee attrition and generate insights that can help organizations improve employee retention.

---

## 🎯 Objective
- To calculate overall attrition rate  
- To identify high attrition areas across job roles and departments  
- To analyze attrition based on demographics and performance  
- To support HR in making better retention decisions  

---

## 🛠️ Tools Used
- Microsoft Excel  
- Pivot Tables  
- Charts (Bar, Pie, Line)  
- Slicers  

---

## 📊 Dataset Details
The dataset contains employee information such as:
- Employee ID  
- Job Role  
- Age  
- Gender  
- Education  
- Department  
- Tenure  
- Performance Rating  
- Attrition Status  

The data was cleaned and prepared before analysis.

---

## 🔧 Data Preparation
- Cleaned column names and formatted data  
- Handled missing and inconsistent values  
- Converted required columns into correct formats  
- Created new columns such as:
  - Age Group  
  - Attrition Flag  
  - Tenure  

---

## 🔄 Data Transformation
- Filtered relevant data for analysis  
- Created calculated metrics such as attrition rate  
- Grouped employees based on performance and age  

---

## 📈 Analysis Performed

### 🔹 Overall Attrition
- Calculated total attrition rate  
- Compared attrition across departments and roles  

### 🔹 Demographic Analysis
- Attrition by age group  
- Attrition by gender  
- Attrition by education level  

### 🔹 Performance Analysis
- Compared attrition across performance levels  
- Identified trends among different employee categories  

### 🔹 Department Analysis
- Identified departments with higher attrition  
- Highlighted areas needing improvement  

### 🔹 Time-Based Analysis
- Observed attrition trends over time  
- Identified peak attrition periods  

---

## 📊 Dashboard
The Excel dashboard includes:
- KPI cards (Total Employees, Attrition, Attrition Rate)  
- Bar charts for job role and department analysis  
- Pie chart for gender-based attrition  
- Column chart for age group analysis  
- Line chart for trend analysis  


---

## 🔍 Key Insights
- Certain job roles and departments have higher attrition  
- Employees with lower tenure tend to leave more  
- Performance level has an impact on attrition  
- Some demographic groups show higher turnover trends  

---

## 💡 Conclusion
This project provides a clear understanding of employee attrition patterns. The insights can help HR teams take data-driven actions to reduce attrition and improve employee retention.
